# Library React

Library React is a responsive online library application built with React. Users can browse books, search titles, apply filters, manage their cart, and navigate through a dynamic shopping experience.


## Live Demo

https://library-react-gold.vercel.app


## Features

- Browse and explore books
- Search books by title
- Filter and sort results
- Add and remove books from cart
- Dynamic routing between pages
- Responsive design


## Tech Stack

- React
- JavaScript
- CSS
- React Router
- Vercel


## What I Learned

This project strengthened my understanding of component architecture, state management, routing, dynamic rendering, and building responsive user interfaces.
